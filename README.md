# Complaint Management System

This is a web-based Complaint Management System project.

## Features
- User Registration & Login
- Submit Complaints
- Track Complaint Status
- Admin Dashboard
- Complaint Status Update

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP
- MySQL
- XAMPP

## How to Run
1. Install XAMPP
2. Copy project to htdocs
3. Start Apache & MySQL
4. Import database
5. Open browser and run project
